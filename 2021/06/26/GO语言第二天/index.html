案例：

以下假设A = 60 , B = 13 ，A的8位二进制：00111100 ，B的8位二进制：00001101

按位与运算符&：

A：   0 0 1 1 1 1 0 0 
B：   0 0 0 0 1 1 0 1
A&B： 0 0 0 0 1 1 0 0 
对应位比较，如果都为1，那么按位与的结果为1，否则为0

按位或运算符|：

A：   0 0 1 1 1 1 0 0 
B：   0 0 0 0 1 1 0 1
A|B： 0 0 1 1 1 1 0 1 
对应位比较，只要有一个为1，那么按位或的结果为1，否则为0

按位异或运算符^：

A：   0 0 1 1 1 1 0 0 
B：   0 0 0 0 1 1 0 1
A^B： 0 0 1 1 0 0 0 1 
对应位比较，只要不相同就为1，否则为0

左移运算符<<

A：   0 0 1 1 1 1 0 0 
A<<2: 1 1 1 1 0 0 0 0
A<<3: 1 1 1 0 0 0 0 0
各二进位全部左移n位,超出总位数就丢弃
在不丢弃的情况下，相当于10进制上乘以了2的n次方

右移运算符>>

A：   0 0 1 1 1 1 0 0 
A>>2: 0 0 0 0 1 1 1 1
A>>3: 0 0 0 0 0 1 1 1
各二进位全部右移n位,超出范围就丢弃
在不丢弃的情况下，相当于10进制上除以了2的n次方
